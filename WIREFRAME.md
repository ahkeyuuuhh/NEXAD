## DESIGN NOTES

### Color Scheme
- **Primary:** Blue (#2563eb) - Actions, links
- **Success:** Green (#10b981) - Completed, confirmed
- **Warning:** Yellow/Orange (#f59e0b) - Pending, reminders
- **Danger:** Red (#ef4444) - Urgent, errors
- **Neutral:** Gray shades - Text, borders
- **AI Indicator:** Purple (#8b5cf6) - AI-related content

### Typography
- **Headings:** Bold, 18-24px
- **Body:** Regular, 14-16px
- **Captions:** 12px
- **Font:** System default (San Francisco iOS, Roboto Android)

### Spacing
- **Container Padding:** 16px
- **Card Spacing:** 12px between cards
- **Button Height:** 48px (touch-friendly)
- **Icon Size:** 24px standard, 20px small

### Interactions
- **Tap Targets:** Minimum 44x44px
- **Loading States:** Skeleton screens or spinners
- **Animations:** 200-300ms transitions
- **Feedback:** Toast messages for actions

### Accessibility
- **Contrast Ratio:** 4.5:1 minimum
- **Text Size:** Scalable (support iOS Dynamic Type)
- **Labels:** All interactive elements have labels
- **Focus Indicators:** Visible for keyboard navigation
